## 🇺🇸 English

**Feed All Animals** is a client-side Minecraft mod that allows you to quickly feed nearby animals using the food in your hand.

### ✨ Features

- **Mass Feeding**: Feeds all animals within a radius of the player with a single key press.
- **Smart Detection**: Only feeds animals that can eat the item you are holding.
- **Client-Side**: Works entirely on the client side.

### 📥 Installation

1. Download and install [Fabric Loader](https://fabricmc.net/use/).
2. Download [Fabric API](https://modrinth.com/mod/fabric-api).
3. Download **Feed All Animals**.
4. Place all `.jar` files into your `mods` folder.

### 🎮 Usage

- Hold the food item in your main hand.
- Press the **G** key (default) to feed nearby animals.
- The keybinding can be configured in Options -> Controls -> Key Binds under the **Feed All Animals** category.

## 🇷🇺 Русский

**Feed All Animals** - это клиентский мод для Minecraft, который позволяет быстро кормить животных поблизости, используя еду в вашей руке.

### ✨ Особенности

- **Массовое кормление**: Кормит всех животных в радиусе от игрока одним нажатием клавиши.
- **Умное обнаружение**: Кормит только тех животных, которые едят предмет у вас в руке.
- **Клиентский мод**: Работает полностью на стороне клиента.

### 📥 Установка

1. Скачайте и установите [Fabric Loader](https://fabricmc.net/use/).
2. Скачайте [Fabric API](https://modrinth.com/mod/fabric-api).
3. Скачайте **Feed All Animals**.
4. Поместите все `.jar` файлы в папку `mods`.

### 🎮 Использование

- Возьмите еду в главную руку.
- Нажмите клавишу **G** (по умолчанию), чтобы покормить животных рядом.
- Клавишу можно настроить в Настройки -> Управление -> Назначение клавиш в категории **Feed All Animals**.
